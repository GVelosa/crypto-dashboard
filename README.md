# Crypto Dashboard

An interactive cryptocurrency dashboard built with **Python** that displays real-time market data using the CoinGecko API.

The application presents cryptocurrencies as **interactive cards**, allowing users to explore price history, market comparisons, and currency conversions through a clean visual interface.

---

## 🚀 Project Goals

This project was created to practice and demonstrate:

* API integration
* Data visualization
* Interactive UI design
* Component-based architecture
* Clean Python project structure

---

## 📊 Features (Planned)

* List of cryptocurrencies displayed as cards
* Expandable card with detailed information
* Historical price charts
* Currency conversion (BRL / USD)
* Market comparison with top cryptocurrencies

---

## 🛠 Tech Stack

* **Python**
* **Flet** (UI framework)
* **CoinGecko API**
* **Requests**

## ⚙️ Setup

Clone the repository:

```bash
git clone https://github.com/your-username/crypto-dashboard.git
cd crypto-dashboard
```

Install dependencies:

```bash
uv sync
```

Run the project:

```bash
uv run python src/crypto_dashboard/main.py
```
